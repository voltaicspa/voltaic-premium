🎓 FORMATOS DISPONIBLES
════════════════════════════════════════════════════════════════════════════════

.txt (Archivo de texto plano)
└─ Abre con: Bloc de notas, VS Code, cualquier editor
└─ Leer en: Navegador, Notepad
└─ Ventaja: Compatible con todo

.md (Markdown)
└─ Abre con: VS Code, GitHub, cualquier editor
└─ Mejor en: GitHub, preview en navegador
└─ Ventaja: Mejor formato visual (títulos, tablas, colores)

.js (JavaScript)
└─ Abre con: VS Code, editor de código
└─ NO abras con: Bloc de notas (se puede pero es incómodo)
